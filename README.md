# Flask Blog App

A clean and minimal blog application built with **Flask**, **Flask-Login**, **Flask-SQLAlchemy**, **Flask-WTF**, **CKEditor**, and **Bootstrap 5**.

This app allows users to:

- Register and login
- Create, edit, and delete blog posts
- Add comments to posts
- Use a rich text editor (CKEditor) with support for images

---

## Features

- **Flask-Login** for user authentication
- **Flask-SQLAlchemy** for database management
- **Bootstrap 5** styling via `bootstrap-flask`
- **CKEditor** for rich text blog content
- Clean and minimal user interface

---

## Installation

1. Prerequisites

Ensure you have uv installed. If not, you can install it via:
Bash

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

2. Clone the repository:

```bash
git clone git@github.com:ronnin796/flask_blog_app.git
cd flask_blog_app
```

2. Run using :

```bash
uv run app.py
```
